## Padrões de Conteúdo Identificados

### Tipo 1: Data Reveal (Carrossel Educativo)

**Formato:** Carrossel com dados de pesquisa de mercado
**Hook típico:** "[Número]% dos compradores de imóveis [insight surpreendente]"
**Exemplo de mensagem:**
"Marketing sem dados é ruído. Marketing com dados é resultado."
"Em [ano], o mercado imobiliário [evento] — entenda o que isso significa para você."

**Estrutura:**
- Slide 1: Hook com dado impactante
- Slides 2-4: Contextualização e análise
- Slide 5-6: Impacto para o comprador/incorporador
- Slide 7: CTA com keyword
